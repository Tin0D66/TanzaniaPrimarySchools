# TanzaniaPrimarySchools
A list of all primary schools in Tanzania. 
Json format
[
  {
  "Name":"region_name",
  "districts" : [
          {
           "Name":"district_name",
           "schools":[
                {
                  "Name":"school_name",
                  "Code":"school_code"
                }
            ]
          }
      ]
  }
]
